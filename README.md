## рџљЂ Getting Started

### Clone the repository

```bash
git clone https://github.com/Dev-Darpan7/DESIAROMA.git
```

### Move the project

Copy the project folder into your XAMPP `htdocs` directory.

```text
C:\xampp\htdocs\DESIAROMA
```

### Import the database

1. Start **Apache** and **MySQL** in XAMPP.
2. Open **phpMyAdmin**.
3. Create a database named `desiaroma`.
4. Import the `desiaroma.sql` file.

### Configure the database

Update the database credentials in `config.php` if necessary.

### Run the project

Open the following URL in your browser:

```text
http://localhost/DESIAROMA
```
